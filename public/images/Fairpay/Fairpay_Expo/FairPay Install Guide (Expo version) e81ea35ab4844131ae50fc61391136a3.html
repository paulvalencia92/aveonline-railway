FairPay Install Guide (Expo version)

1. Requirement

1.1. Setup Android Studio IDE (for Android developer)

Follow Android Studio official page to install https://developer.android.com/studio/install

For this version, we are using Android Studio Arctic Fox | 2020.3.1 Patch 2

After install Android Studio IDE, we need create virtual device.

Open Android Studio IDE, select AVD Manager

Click Create Virtual Device...

Then download API 30, create a virtual device like image above

1.2. Setup xCode IDE (for iOS developer)

Follow Apple official page to install https://developer.apple.com/xcode/

1.3. Install NodeJS

For easy change and update NodeJS version, we will use NVM.

https://github.com/nvm-sh/nvm

⚠️ You can install directly with NodeJS official page https://nodejs.org/en/download/

After install, check version with command below. Please use NodeJS with 14.x version and NPM is 6.x version.

$ node --version
v14.17.5

$ npm --version
6.14.14

1.4. Install yarn

Follow Yarn official page to install https://classic.yarnpkg.com/en/docs/install/

After install, check version with command below. Please use version 1.22.11+

$ yarn --version
1.22.11

1.5. Install expo-cli

Follow Expo official page to install https://docs.expo.dev/workflow/expo-cli/

After install, check version with command below. Please use version 4.11.0+

$ expo-cli --version
4.11.0

2. Run project

Please extract project source code, then use terminal to access project source code directory.

$ cd <project-source-code-directory>

Use list command, we can see project files like image below.

Then, run yarn command to install NodeJS modules packages

$ yarn

Finally, we will start project with iOS simulator or Android virtual device with command below.

$ yarn ios
$ yarn android
